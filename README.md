# Idea Box Website

This project is a simple idea box application built with React.js and styled using Tailwind CSS. Users can submit their ideas, view a list of submitted ideas, and navigate through different pages.

## Features

- Submit new ideas
- View a list of submitted ideas
- Responsive design using Tailwind CSS

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd idea-box-website
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the development server, run:
```
npm start
```

The application will be available at `http://localhost:3000`.
